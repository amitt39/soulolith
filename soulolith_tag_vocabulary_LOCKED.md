# Soulolith Master Product Tag Vocabulary
> Version: LOCKED FINAL | Total tags: 171
> Rule: NEVER use any tag outside this vocabulary
> Format: lowercase, hyphen-separated only
>
> Prefix logic:
> crystal-   = what stone it is made of
> meaning-   = what property/benefit it carries
> chakra-    = which chakra it activates
> type-      = what form/shape the product is
> decor-     = home decor category
> spiritual- = spiritual/ritual item category
> material-  = what material it is made of (composition)
> intent-    = energy/protection intent product category
> gift-      = gifting suitability
> zodiac-    = zodiac sign suitability

## TAGGING RULES
1. NEVER create a tag outside this vocabulary
2. NEVER use marketing tags (sale, new, trending, featured etc)
3. ALWAYS reuse existing tags — never invent variations
4. Use lowercase only, hyphen-separated only
5. Every product must have at minimum:
   - 1 crystal- tag (multiple if it is a set)
   - 1 type- OR decor- OR spiritual- OR material- tag
   - 3–6 meaning- tags
6. Add chakra-, zodiac-, gift- tags wherever relevant
7. Target 8–12 tags per product for maximum collection coverage
